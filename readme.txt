This sidebar widget displays your Stack Overflow flair. Configurable title, theme, and user ID.

== Installation ==

1. Upload the `stackoverflow-flair-widget` directory into your `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the widget to your sidebar through the 'Widgets' panel under 'Appearance'.
4. Configure the widget. At minimum, you should specify your User ID. If you're not sure what your
user ID is, login to Stack Overflow and view your profile. Your user ID is the number in the URL.
5. Don't forget to save the configuration.
